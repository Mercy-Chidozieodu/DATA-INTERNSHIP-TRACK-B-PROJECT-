**IBM HR Attrition Analysis Project**
 📌 Project Overview
This project analyzes employee attrition using data analytics tools to uncover key factors driving workforce turnover. The analysis focuses on identifying trends in employee satisfaction, promotion stagnation, income distribution, and other HR metrics.

** 🎯 Objectives**
- Analyze employee attrition patterns  
- Compare satisfaction levels across employees  
- Identify promotion stagnation trends  
- Examine income distribution  
- Evaluate impact of stock options on attrition  

---
**🛠 Tools Used**
- Excel (Data Cleaning , Pivot Tables and slicers)  
- Tableau (Data Dashboard)  
    

 📊 **Key Analysis Performed**
- Attrition cross-tab analysis  
- Satisfaction comparison  
- Income distribution analysis  
- Promotion stagnation analysis  
- Stock option vs attrition analysis  
- Interactive dashboard with filters  

📈 **Key Insights**
- High attrition is concentrated among employees with low satisfaction levels  
- Employees with longer years without promotion show higher turnover risk  
- Lower income groups experience higher attrition rates  
- Stock options appear to reduce attrition likelihood  

📂 **Project Files**
- Excel Analysis File  
- Tableau Dashboard (.twbx)  

## ✅ Conclusion
This analysis highlights key drivers of employee attrition and provides insights that can help organizations improve retention through better compensation, career growth opportunities, and employee engagement strategies.
